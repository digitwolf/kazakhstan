# Per-day ukiyo-e image prompts (for Codex / any image model)

Generate each as a horizontal **1200×500** image, traditional Japanese **ukiyo-e woodblock** style
(Hokusai/Hiroshige), flat colour, bold outlines, mulberry-paper texture, limited indigo/vermilion/ochre
palette, no text. Save as `assets/day-<N>.png`. The site auto-uses the PNG once it's there.

Common style suffix to append to every prompt:
> *…ukiyo-e woodblock print, Hokusai/Hiroshige style, flat colours, bold outline, washi paper texture, no text, horizontal composition.*
