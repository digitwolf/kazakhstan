# Interests Library — what this family loves, mapped to the PNW first tour

This library is **interest-driven**, not logistics-driven. Where [`../research/`](../research/README.md)
answers *"is the route safe, open and rideable?"*, this answers *"given what Ruslan, Galiya and Aslan
actually love, what's the win on each day?"* Read [`../tour/00-family.md`](../tour/00-family.md) for
the full profiles; the themes here come straight from that file's "what we like" checklist.

This is a 7-day **domestic loop on home roads**, so unlike the old Japan version there's no
"off-route / bookend" problem — **almost everything below is on or just off the planned route.**
Re-confirm any hour/price/closure before relying on it; sources are linked per item.

## The triple-win stops (hit several people at once)

A strong stop serves more than one person — these are the standouts:

1. **Tillamook (Day 3)** — Creamery (Galiya food + Aslan ice cream/toys) **and** the Air Museum next door (Aslan + Ruslan). The single best multi-interest stop of the trip.
2. **Yachats rest day (Day 4)** — Heceta Head + Thor's Well (Galiya), Sea Lion Caves + tidepools + fireworks (Aslan), a splurge dinner (Galiya), bikes parked (everyone).
3. **Astoria (Day 2)** — the Column + gliders + Maritime Museum (Aslan + Ruslan) and Bowpicker fish & chips (Galiya).
4. **Day 7 finale** — Paradise/Reflection Lakes scenery (Galiya), snow + waterfalls + wildlife (Aslan), and the Chinook Pass graduation ride (Galiya + Ruslan). The emotional close.

## How agents should use this
- **tour-expert:** weight **Galiya's coast food + lighthouses/scenery** and **Aslan's wildlife/hands-on/toy** stops when choosing or adding stops, alongside the safety/pace rules; favor options that hit several interests at once (the triple-wins above). Never invent travel facts — add them to `tour/` first.
- **website-builder:** surface these interests in framing/emphasis (foodie picks, lighthouses & scenery, kid wildlife & toys, and the "new rider, road-as-teacher" graduation arc) — but render only what's in `tour/`.
